# UniBurguer – Cardápio Web com Carrinho

Projeto acadêmico para a disciplina de Usabilidade, criando uma página web de cardápio com carrinho funcional e persistente.

## Funcionalidades
- Carrinho funcional: adicionar, remover e alterar quantidade de itens  
- Persistência com LocalStorage  
- Cálculo automático de valores  
- Design responsivo para mobile  
- Página de pagamento com resumo do pedido  

## Tecnologias
HTML5 | CSS3 | JavaScript | LocalStorage

## Execução
1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador  
3. Teste as funcionalidades do cardápio e do carrinho
